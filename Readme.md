**Project Title:**

### Enterprise Budget vs Actual \& Forecast Analysis (PBCS-Aligned)



**Business Objective:**

###### Analyze Budget, Actual, and Forecast data to identify overspending, evaluate forecast accuracy, and provide executive-level insights for better planning decisions.



**Tools Used:**



* SQL (data preparation \& variance logic)



* Excel (validation \& FP\&A-style analysis)



* Power BI (management reporting dashboards)



* Tableau (executive storytelling \& pattern detection)





**Key Analysis Performed:**



* Budget vs Actual variance analysis



* Expense overspend identification by department



* Forecast accuracy evaluation over time



* Entity-level performance comparison





**Key Insights:**



* IT and Sales departments showed consistent overspending across entities



* APAC entity contributed the highest unfavorable variance



* Forecast accuracy improved after mid-year reforecast cycles





**Outcome:**

Delivered a consultant-style FP\&A dashboard aligned with Oracle PBCS planning concepts.
